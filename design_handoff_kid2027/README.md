# Handoff: Kanagawa Innovators Day 2027（KID 2027）ウェブサイト

## Overview
神奈川県主催「Kanagawa Innovators Day 2027（KID 2027）」公式サイトのワイヤーフレーム一式（3ページ）。前回大会（KID 2026 / pref.kanagawa.jp の kid2026 配下）の構成・内容を踏襲し、2027年版として開催情報を更新したもの。ヘッダー／フッターは全ページ共通で、ヘッダーナビから相互に遷移する。

## About the Design Files
このバンドル内の `.dc.html` は **HTMLで作成したデザインリファレンス（ワイヤーフレーム）**です。意図したレイアウト・情報設計・挙動を示すプロトタイプであり、そのまま本番コードにコピーするためのものではありません。タスクは、これらのデザインを **対象コードベースの既存環境（React / Vue / Next.js / 静的サイトジェネレータ等）の確立されたパターン・ライブラリで再現する**こと。環境が未定なら最適なフレームワークを選定して実装してください。各ファイルは軽量なコンポーネントランタイム上で動作し、ブラウザで直接開けます（ロジックはセッションページのフォトスライダー相当の状態のみ）。

## Fidelity
**ワイヤーフレーム（low-fi寄り）**。レイアウト・情報設計・セクション順・コピー・遷移は確定に近い精度だが、視覚表現は未確定。以下は意図的なプレースホルダー：

- 画像・ロゴ・登壇者写真・地図・フロアマップ・アイコン・イラスト・出展社ロゴは破線ボックス（`.wf-ph`）。
- 実績数値（来場者数 `0,000名` / 出展満足度 `00%` / 来場者属性グラフ）はダミー。
- ボタンはワイヤー用のモノトーン（塗り `.wf-btnf` = #3a3a3a / 枠線 `.wf-btn`）。本番はブランドカラーに置換。
- キービジュアル画像（`assets/kv.jpg`）のみ実画像。開催日・会場が画像内に描かれているため、KV周辺にテキストは重ねていない。
- 「STARTUP PITCH」ページは今後追加予定。**ヘッダーナビからは意図的に除外済み**（リンクを追加しないこと）。

実装時は、レイアウト・フロー・コピー・遷移はデザイン通りに再現しつつ、**スタイリングは対象コードベースの正式なデザインシステム**（本番の色・タイポ・コンポーネント）を適用。破線プレースホルダーは本番アセットに差し替える。

## Page 2: セッション / タイムテーブル
ヘッダー → ページ見出し（SESSION / TIMETABLE）→ STAGE INFORMATION → コントロール（DAY切替＋凡例）→ タイムテーブル → フッター。

- **STAGE INFORMATION** — 5ステージのテーマ・会場/定員（仮）カード（5列、SPで2→1列）。
- **DAY切替タブ** — DAY1（1.21木・STARTUP PITCH NIGHT）/ DAY2（1.22金・MAIN SUMMIT）。選択タブは塗り、他は透明。
- **凡例** — 現在時刻（赤ライン）/ 開催中（赤枠）/ 予定（破線枠）/ 終了（グレー）。
- **タイムテーブル＝テレビ番組表スタイル**（最重要）：
  - **縦列＝ステージ（5列）／横行＝時間軸**。左端に時刻軸（1時間ごとラベル）、上端にステージ見出し（各ステージを色分け）。
  - CSS Grid で構成。グリッド行は30分＝1スロット（現状 `SLOT=78px`）、ヘッダー行 `HEAD=40px`。列は `64px repeat(5, minmax(128px,1fr))`。
  - 各セッションは開始・終了時刻から `grid-row` を算出して配置（`rowLine = (分−基準分)/30 + 2`）、`grid-column = ステージindex+2`。
  - セッションカード：タイトル（2行clamp）＋時間＋登壇者（1行clamp）。
  - **現在開催中の表示**：仮の現在時刻（DAY2＝15:20）を基準に、
    - グリッド全幅に**赤い横ライン（NOWバー）**を絶対配置（top = `HEAD + (現在分−基準分)/30 * SLOT`）。左端に「NOW 15:20」ラベル。
    - 現在時刻に**かかっているセッション**＝赤枠＋赤系シャドウでハイライト。
    - **終了したセッション**（現在時刻より前に終了）＝グレースケール＋不透明度ダウン、右上に「終了」バッジ（タイトルはバッジ分の右余白を確保）。
    - DAY1は過去日想定でNOWバー無し。
  - 横スクロール可（`min-width:760px`）。
  - 内容（ステージ名・セッション）は仮。データはロジック内 `data(day)` の配列で定義（前回登壇者を織り込んだダミー）。

### State（セッション）
- `day`（1 or 2, 初期2）：表示日。切替で `data(day)` を差し替え、グリッド・セッション・NOWバーを再計算。
- レイアウト定数 `SLOT`（スロット高px）/ `HEAD`（ヘッダー高px）はグリッド行高とNOWバー位置計算で共有。本番でも「30分スロット高×行番号」の整合を必ず保つこと。
