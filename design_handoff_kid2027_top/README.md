# Handoff: Kanagawa Innovators Day 2027（KID 2027）トップページ

## Overview
神奈川県主催「Kanagawa Innovators Day 2027（KID 2027）」の公式サイト・トップページ。前回大会（KID 2026 / pref.kanagawa.jp の kid2026 ページ）の内容を踏襲し、2027年版として開催情報を更新したもの。1枚のロングランディングページで、イベントのコンセプト・前回実績・コンテンツ・登壇者・協賛企業・開催概要までを縦スクロールで見せる。

## About the Design Files
このバンドル内の HTML ファイルは **HTML で作成したデザインリファレンス**です。意図した見た目・挙動を示すプロトタイプであり、そのまま本番コードにコピーするためのものではありません。タスクは、これらの HTML デザインを **対象コードベースの既存環境（React / Vue / Next.js / 静的サイトジェネレータ等）で、その確立されたパターン・ライブラリを用いて再現する**ことです。環境がまだ無い場合は、プロジェクトに最適なフレームワークを選定して実装してください。

- `KID2027-top.dc.html` — 元デザイン（軽量なコンポーネントランタイム上で動作。ロジックはフォトスライダーのみ）。構造・スタイルの参照用。
- `KID2027-top-standalone.html` — フォント・画像を全てインライン化した自己完結版。ブラウザで直接開いて挙動確認できる（オフライン可）。
- `assets/kv.jpg` — キービジュアル画像（ヒーロー部）。

## Fidelity
**Low-fidelity 寄りのワイヤーフレーム**です。レイアウト・情報設計・セクション順・コピーは確定に近い精度ですが、**視覚表現はワイヤーフレーム段階**です。以下は意図的にプレースホルダーになっています：

- 画像・ロゴ・登壇者写真・地図・アイコン・コンセプトビジュアル・イラストは、破線ボックスのプレースホルダー（`.wf-ph`）。
- 「来場者数」「出展満足度」「来場者属性」の数値・グラフはダミー（`0,000名` / `00%` / 棒グラフ）。確定値に差し替える。
- キービジュアル（`assets/kv.jpg`）のみ実画像が入っている。KV 内に開催日・会場が描かれているため、KV 周辺にはテキストを重ねていない。

実装時は、レイアウト・フロー・コピーはこのデザイン通りに再現しつつ、**スタイリングは対象コードベースの正式なデザインシステム**（本番の色・タイポグラフィ・コンポーネント）を適用してください。破線プレースホルダーは本番アセットに置き換えます。

## Interactions & Behavior
- **ナビ・CTA**：全てページ内アンカー（`#report` `#contents` `#partners` `#outline` 等）へのスムーズスクロール（`html{scroll-behavior:smooth}`）。
- **フォトスライダー**：左右ボタンで `photo` インデックスを 0→3 の範囲で増減（6枚 / 3枚表示 = 有効位置4つ）。トラックを `translateX` で移動、ドットが現在位置を反映。ループなし（端でクランプ）。
- **レスポンシブ**：`@media(max-width:900px)` で 2カラム→縦積み、6列→3列、ナビ→ハンバーガー。`@media(max-width:560px)` で 6列→2列、3列→1列。
- **ハンバーガーメニュー**：表示のみでトグル未実装。実装側でドロワー/オーバーレイ開閉を追加すること。
- ホバー：リンク color 変化（#2a63d6 → #1a49a8）のみ定義。ボタンのホバー状態は要追加。

## State Management
- `photo`（整数, 初期0, 範囲 0–3）：フォトスライダーの現在位置。`prev`（`max(0, photo-1)`）／`next`（`min(3, photo+1)`）で更新。トラックの transform とドットの色がこの値から導出される。
- 他に動的状態なし。ハンバーガー開閉状態は実装時に追加（例：`menuOpen: boolean`）。
